Build instructions

Budiling a portlet WAR, step by step.

  • Run ant from the webapp project directory
  • cd to webapps directory
  • To build a portlet webapp, run: ant build-portlet
  • Use the container system property to target a specific container, e.g. '-Dcontainer=exo'.
    • Several containers are supported.
    • Example command:
  • Check the etc/yourcontainer directory for a README file. If available, check for further instructions.
  • Deploy the WAR to your portal server
